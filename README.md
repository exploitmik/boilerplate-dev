# boilerplate-dev
Estrutura inicial de um projeto estático.

# Tecnologias utilizadas
* webpack
* postcss
* cssnext
* scss
* cssnano
* mini-css-extract-plugin
* terser-webpack-plugin
* babel
* eslint

# Como usar

## Instalar todas a dependências
```
npm install
```

## Rodar em modo de desenvolvimento
```
npm run dev
```

## Rodar o build
```
npm run build
```
